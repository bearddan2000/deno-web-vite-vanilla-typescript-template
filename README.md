# deno-web-vite-vanilla-typescript-template

## Description
A vanilla template for the vite packager.

## Tech stack
- deno
- vanilla
- typescript
- vite

## Docker stack
- archlinux

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
